## **Опис проекту**

У цьому репозиторії представлено виконання практичного завдання з написання DQL (Data Query Language) запитів у MySQL. Завдання полягало у вибірці, фільтрації, сортуванні та агрегації даних із завантаженої бази даних.

## **Вміст репозиторію**

* SQL Commands.txt — Файл із текстом усіх виконаних SQL-запитів.  
* p1\_1\_select\_all.png — Результат вибірки всіх стовпчиків з таблиці products.  
* p1\_2\_select\_shippers.png — Результат вибірки стовпчиків name та phone з таблиці shippers.  
* p2\_aggregate.png — Результат використання агрегатних функцій (AVG, MAX, MIN) для ціни товарів.  
* p3\_distinct\_limit.png — Результат вибірки унікальних значень (category\_id, price) із сортуванням за спаданням та обмеженням у 10 рядків.  
* p4\_count\_between.png — Результат підрахунку кількості продуктів у ціновому діапазоні від 20 до 100\.  
* p5\_group\_by.png — Результат групування даних за постачальником (supplier\_id) з підрахунком кількості продуктів та їхньої середньої ціни.

## **Інструменти**

* MySQL / MySQL Workbench  
* DQL (Data Query Language: SELECT, WHERE, GROUP BY, ORDER BY, LIMIT, агрегатні функції)
